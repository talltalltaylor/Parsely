Parsely

Browser Extension that parses HTML cooking pages into formatted, abridged recipes. Usage:
  • Click the Parsely logo, top button, while viewing one of the sites in our domain list to parse a recipe, if one is present.
  • The Parsely extension is curruntly working on the following websites:
  • Clicking the settings icon, the middle button, will open the optional settings. Click it again to close settings.
  • Clicking the help button, the bottom button, will open this page in a seperate tab.
